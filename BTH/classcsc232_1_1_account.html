Detailed Description

An Account object represents any kind of bank account that maintains a current balance and allows deposits, withdrawals, and transfers to other accounts. It is an abstract class, because any extending class needs to specify what happens at the end of the month (e.g., fees or interest payments).

For simplicity, the account balance is stored as an integer.
